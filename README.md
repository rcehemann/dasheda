# DASHEDA

## interactive exploratory data analysis in Plotly+Dash

#### ROADMAP:
- support for generated (a la `sklearn.datasets`) and real data
- slicing, normalization, and dimensionality reduction
- prototyping of simple predictive models

#### REQUIREMENTS
- python 3.7
- dash 0.2.1
- plotly 3.4.2
- scikit-learn 0.20.1
